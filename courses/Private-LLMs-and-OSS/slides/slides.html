The Neuron

  • The fundamental "unit" of an LLM
  • Input: Receives data from other neurons
  • Weight: Decides how much that input "matters"
  • Activation: Fires if the signal is strong enough
Sum + Bias Σ

Math of Weights

  • Every "connection" is a floating-point number.
  • Input x is multiplied by weight w
  • Output: y = σ(wx + b)
  • Billions of these happen per token

Weight Matrix (W)

[ 0.01, -0.23, 0.45 ]
[ 0.88, 0.12, -0.01 ]
[ -0.42, 0.99, 0.07 ]

VRAM Bottleneck

  • Weights ⇒ Parameters
  • A 7B model has 7 billion parameters
  • At FP16 (Full precision):
  • 7B * 2 bytes = *14GB VRAM*
  • Most consumer GPUs have 8GB - 12GB

Quantization

  • Compressing weights to fit your hardware
  • Convert 16-bit floats to 4-bit integers
  • Reduces memory from 14GB ⇒ ~5GB
  • Minimal loss in performance
  • Modern LLM viability

Attention

  • Self-Attention: Weights that score word relevance
  • Feed Forward: Processing the context
  • Weights define the "knowledge"
  • "Training" refers to deriving these weights
