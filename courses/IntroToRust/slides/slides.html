The Type System

  • Algebraic Types
  • Lets the compiler do the work
  • Extensive Enums

Enums

  • NOT just named integers
  • Compiler knows every enum type
  • Provable
  • "Sum Type"

Classic Example

enum Direction {
    North,
    South,
    East,
    West,
}
  • The compiler knows that if you have a Direction
  • It must be one of those 4 types
  • Sum of the number of enumerations

Structs

  • Groups related data together
  • Non mutually-exclusive
  • "Product Type"

Simple Color

struct Color{
  r: bool,
  g: bool,
  b: bool,
}

let black = Color { r: false, g: false, b: false };
let blue  = Color { r: false, g: false, b: true };
let white = Color { r: true,  g: true,  b: true };
  • If you have a Color:
  • It must be one of those 8 types
  • Product of the number of enumerations

enum Result<T, E> {
    Ok(T),
    Err(E),
}

Error Handling

fn divide(a: i32, b: i32) -> Result<i32, String> {
    if b == 0 { 
        return Err("cannot divide by zero".to_string()); 
    } 
    Ok(a / b)
}

fn main() {
    let x = match divide(10, 2) {
        Ok(result) => result * 2,
        Err(err) => 0,
    };
}

Chaining

fn divide(a: i32, b: i32) -> Result<i32, String> {
    if b == 0 { 
        return Err("cannot divide by zero".to_string()); 
    } 
    return Ok(a / b);
}

fn do_math() -> Result<i32, String> {
    let temp = divide(15, 3)?; 
    return Ok(temp);
}
  • ? → early return if Err()
  • ? → get value if Ok()

Memory Management

  • No garbage collector
  • No manual free()
  • Ownership & Borrowing system

Ownership Rules

  • Each value has one owner
  • Ownership is transferred on assignment
  • When the owner goes out of scope → value freed

Ownership Ideas

fn greet(name: &String) {
    name = &name.to_uppercase();
    println!("Hello, {}!", name);
}

fn main() {
    let name = String::from("Bob");
    greet(&name);
    println!("You greeted {}", name);
}
    
fn greet(name: &mut String) {
    *name = name.to_uppercase();
    println!("Hello, {}!", name);
}

fn main() {
    let mut name = String::from("Bob");
    greet(&mut name);
    println!("You greeted {}", name);
}
  • Opt in to `mut`
  • Call site changes
  • 1 Mutable reference
  • ∞ *Immutable* references
  • 1 Owner
